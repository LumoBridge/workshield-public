<h2 style="margin:0 0 10px;">Executive Summary</h2>

<p style="margin:0 0 10px;">
  <strong>WorkShield</strong> is a documentation and evidence-preservation platform designed to help individuals and organizations record workplace incidents accurately and consistently over time.
</p>

<p style="margin:0;">
  The product emphasizes <strong>structure, integrity, and user control</strong>. It does not provide legal advice, determine outcomes, or replace professional judgment.
</p>

</div>

<br />

<h2 style="margin:0 0 10px;">Problem</h2>

<div style="
  border-left:4px solid #22c55e;
  padding-left:14px;
  margin:0 0 16px;
">

<p style="margin:0 0 10px;">
  Workplace incidents are often documented late, inconsistently, or across fragmented tools. This results in lost context, incomplete records, and poor preparedness when information is eventually needed.
</p>

<p style="margin:0;">
  Organizations require a neutral system that prioritizes <strong>factual capture, traceability, and intentional sharing</strong> without legal positioning or risk exposure.
</p>

</div>

<hr style="border:none; border-top:1px solid #1e293b; margin:18px 0;" />

<h2 style="margin:0 0 10px;">Solution</h2>

<div style="
  border:1px solid #1e293b;
  border-radius:10px;
  padding:16px;
  background:#020617;
">

<p style="margin:0 0 10px;"><strong>WorkShield provides</strong></p>

<ul style="margin:0 0 10px; padding-left:18px;">
  <li>Structured incident documentation workflows</li>
  <li>Secure, centralized storage with integrity controls</li>
  <li>Guided prompts and educational context</li>
  <li>Intentional, user-controlled exports</li>
</ul>

<p style="margin:0;">
  The platform is designed to support preparedness and internal clarity while maintaining strict scope boundaries.
</p>

</div>

<hr style="border:none; border-top:1px solid #1e293b; margin:18px 0;" />

<h2 style="margin:0 0 10px;">Product Structure</h2>

<h3 style="margin:0 0 10px;">Individual Plans</h3>

<div style="border:1px solid #1e293b; border-radius:10px; padding:14px;">

  </ul>

<br />

<h3 style="margin:0 0 10px;">Managed Access (Organizations)</h3>

<div style="border:1px solid #1e293b; border-radius:10px; padding:14px;">
  <ul style="margin:0; padding-left:18px;">
    <li>Teams of two or more</li>
    <li>Shared workspaces with governance</li>
    <li>Role-based access and permissions</li>
    <li>Controlled, auditable exports</li>
    <li>Advanced capabilities included by default</li>
  </ul>
</div>

<p style="margin:12px 0 0;">
  This separation reflects real-world usage patterns and reduces risk associated with mixed personal and organizational workflows.
</p>
